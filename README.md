# Random #

Este repositorio contiene un archivo de ejemplo con NodeJs, que apartir de un numero aleatorio en 0 y 1 devuelve por consola
es mayor o menor que 0.5.

//Math.random devuelve un numero aleatorio entre 0 y 1.

# Modo de uso #

node random.js
